Device

Representation of a device in a datacenter

Attributes

  • name:string - Name of the device
  • ename:string - Extended Name of the device
  • type:string - Type of the Device

Methods

  • addToHardware() - Create a Device
  • create() - Create a Device
  • disable() - Disable Device and its hardware
  • enable() - Enable Device to be used.

  • provision() - Provision Hardware to meet the requirements
  • reserve() - Reserve a resource
