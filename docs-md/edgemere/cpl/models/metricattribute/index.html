MetricAttribute

Attribute type of Metric. This is for metrics that are descriptiive not consumable

Attributes

  • name:string - This is the name of the metric
  • value:string - This is the value of the metric

Methods

  • consume() - Consume attribute metric to the current value
  • copy() - Copy Metric
  • lessThanEq() - Test less than or equal to the value passed in
  • minus() - Add attribute metric to the current value
  • plus() - Add attribute metric to the current value
  • valueToString() - Change to string Metric Consumeable
  • factory() - Create a metric based on the metrictype table
  • greaterThanEq() - Test greater than or equal to the value passed in
  • set() - Add value to a metric
