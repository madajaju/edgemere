MetricComposite

Composite type of Metric. This is for metrics that are a group of metrics

Attributes

  • name:string - This is the name of the metric
  • value:string - This is the value of the metric

Associations

Methods

  • consume() - Substract metric to the current value up to zero
  • copy() - Copy Metric
  • create() - Create Composite Metric
  • findDeep() - Find Metric with name
  • findMatchDeep() - Find Metric with name
  • lessThanEq() - Test less than or equal to the value passed in

  • plus() - Add metric to the current value
  • set() - Set metric to the current value
  • factory() - Create a metric based on the metrictype table
